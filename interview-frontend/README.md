# Interview Frontend

React + Vite frontend for the AI Interview Platform.

## Run

```powershell
cd interview-frontend
npm install
npm run dev
```

App URL: `http://localhost:5173`

## Build

```powershell
npm run build
```

## Backend API

- Reads API base from `VITE_API_BASE_URL`
- Defaults to `/api`
- Vite proxy config is in `vite.config.js`
